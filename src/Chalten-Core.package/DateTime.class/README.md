A DateTime is date with a time object in a certain time zone. If no time zone are provided when a DateTime is about to be created, its asume the local one.

Instance Variables
	date:			<ChaltenDate>
	timeOfDay:		<TimeOfDay>
	zone:			<ChaltenTimeZone>

date
	- The date.

timeOfDay
	- The time of day.

zone
	- The time zone.
